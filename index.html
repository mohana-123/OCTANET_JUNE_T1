Building tools

HTML/CSS, Javascript

HTML and CSS are the building blocks of web pages. HTML provides structure, while CSS adds style and layout. JavaScript is a programming language that adds interactivity and dynamic elements to websites. Together, they create the foundation for modern web development.

React Native

React Native is a popular JavaScript framework for building mobile applications. It allows developers to create cross-platform apps using a single codebase, leveraging the power of React to build native user interfaces. With its efficiency and flexibility, React Native has gained widespread adoption in the mobile development community.

Java

Java is a versatile, object-oriented programming language widely used for developing applications. Known for its platform independence, it allows code to run on different operating systems. Java's robust libraries, garbage collection, and wide community support make it popular for building everything from mobile apps to enterprise-level software systems.
